# fokus - 电子学习没有手机

![](http://static.myseu.cn/cgmf5y1496388232.jpeg)

## 简介

这是期末复习玩了一整天手机之后花半小时写出来的小应用，它也许能胜过市场上大多数正念训练应用，帮助你专心学习。

## 理念

这款应用遵循只做一件事的原则，它：

- 只有一个功能：计时
- 只有一个开关：反扣手机
- 只有一个亮点：心跳效果（需要 iPhone 6S 及以上支持 3D Touch 的设备）
- 没有任何多余的文案和界面

## 后续

- 会考虑接入 HealthKit 记录正念时间
